# *MOTRIX*
microcontroller based expansion board and motor driver for beginner robotics freaks.
The board is especially created for making car projects _(e.g. line following, obstacle avoiding, remote control, ...)_ easier and even just _"plug-in and code"_. It integrates motor control, component interfaces, optional wireless communication on a single PC, and it's software side makes coding easier.


## _THE PROBLEMS IT SOLVES_
- **WIRING CHAOS**
- **SOFTWARE & CONTROL LIMITATIONS**
- **POWER MANAGEMENT**

## _FEATURES_
- dual DC motor control using _TB6612FNG_
- Interfaces for common robotics sensors
